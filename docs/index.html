Blink is a PixInsight process developped by Pleiades Astrophoto S.L. and Nikolay Volkov.
This update package installs a second Blink process featuring some minor UI improvements: select all checked images, select all images, select none and invert selection.

Sources are available under: https://github.com/vahlkar/PCL.

Repository URLs:

  • For PixInsight versions 1.8.9-2 build 1579 and later: https://pixinsight.arcturus.ch/blink2/
  • For previous PixInsight versions: https://pixinsight.arcturus.ch/blink2/legacy/

Installation

  • in PixInsight, open menu Resources -> Updates -> Manage Repositories;
  • click Add and input "https://pixinsight.arcturus.ch/blink2/" for PixInsight versions 1.8.9-2 build 1579 and later or "https://pixinsight.arcturus.ch/blink2/legacy/" for older versions, then click OK on both windows to validate;
  • open menu Resources -> Updates -> Check for updates;
  • select Blink2 in the list and validate installation; the module gets installed after PixInsight restart;

Usage

  • go to Process -> ImageInspection -> Blink2
  • use blink normally; 4 buttons are added: select all images, select none, invert selection and select checked images
